Ultimate Accelerated Mobile Pages (AMP) is a Plugin to Enable AMP Version for your website. Feature rich and easy Customizable Plugin.

== Description ==
Ultimate AMP is based on [AMP project](https://www.ampproject.org/). AMP will bring your website faster loading speed and many features like Social Shares, Analytics, Google Adsense, Comments. 

Ultimate AMP has very Powerful Option Panel, easy to use. Enable AMP Version of your website by clicking "Enable" AMP Version.
You're website will load with default Template comes with this Plugin, we'll include gradually with features and Templates.

By Enabling AMP Version your websites All Posts will have AMP-Compatible and posts/pages can be acccessible by appending '/amp/' at the end of your Post/Page URLs.

Example: This is our Blog Post URL is `https://jeweltheme.com/wordpress-theme-development-resources/`, it can now access by AMP Version at `https://jeweltheme.com/wordpress-theme-development-resources/amp/`. 

Bring the speed and features of the open source [AMP project](https://www.ampproject.org/) to your site, the WordPress way.
If you don't have [pretty permalinks](https://codex.wordpress.org/Using_Permalinks#mod_rewrite:_.22Pretty_Permalinks.22) enabled, you can do the same thing by appending `?amp=1`, i.e. `http://example.com/?p=123&amp=1`

== Installation ==

1. Upload the Downloaded Folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You may need to refresh your permalinks by going to `Settings > Permalinks` and tapping the `Save` button.
